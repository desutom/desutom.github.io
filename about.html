Technical Info & Privacy

This website currently runs on an Oracle free-tier VPS. It's served using dockerized nginx on my own domain, with SSL certs provided by Let's Encrypt.


  • This website consists of pure HTML and CSS. No cookies or other tracking methods are used (this is just a simple static website afterall..).
  • Requests may be logged by the webserver itself and contain the client's IP address and time of request. This data is not shared to third-parties of any kind.
  • Keep in mind that requests have to pass through Oracle's network infrastructure.
  • There is no DDOS guard, such as Cloudflare being used, though if for some reason I'll need it I'll have no other choice than to use Cloudflare's services.
